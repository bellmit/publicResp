DEPENDENCIES
------------
Jasper Reports requires the following:

Commons: 
  beanutils 1.7 or later (we use 1.8)
  collections 2.1 or later (we use 3.2.1)
  javaflow sandbox version
  logging 1.0 or later (we use 1.0.4)

JRXML:
  commons-digester 1.7 or later (we use 1.8)
  JDK 1.4, or, Eclipse JDT compiler, or, jikes compiler, or Groovy, or BeanShell
   (we rely on JDK itself)

Report Output:
  PDF: iText 1.3.1 or later (we use 2.1.5)
  XLS (not used): Jakarta POI 3.0.1 or later, JExcelApi 2.6 or later
  Charts: JFreeChart 1.0.0 or later, JCommon required by JFreeChart, 
    Batik SVG Toolkit 1.7 or later if rendering charts as SVG
