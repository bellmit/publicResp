Insta In Patients API

This document defines the API endpoints for incoming patient lists.

IP List

Get IP list

This endpoint gives list of all IP patients with their details.

REQUEST

GET

http://localhost:8080/instahms/inpatients/list.json?status=&page_num=&page_size=&sort_order=&sort_reverse=&search_id=&find_string=
Parameters
page_num int

1 (which page number to fetch)
page_size int

20 (number of items in a page)
sort_order String

“mr no” (name of the field on which to sort)
sort_reverse Boolean

true (sort in ascending or descending; will only work if sortOrder is specified)
search_id int

10 (id of the search name, when present ignore query params get the patient list based on this parameter)
find_string String

“Anup” (search by mrno or patientname or patient phone)
