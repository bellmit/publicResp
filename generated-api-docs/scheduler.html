Search Resources

GET

Parameters
page_num number 1
Default
1

Which page number to fetch
page_size number 20
Default
15

Number of items in each page
sort_order string resource_name

Name of field to sort on
with_count boolean false
Default
false

Whether count of number of records are required
find_string string Dr. sairam

Search input entered by user
departments array DEP0026,DEP0027

Resources should belong to either of these departments
availability_days array 0,1,2,3

Resources should be available in these days
centers array 1,2,3

Resources should be associated with these centers
availability_start_date string 10:00

Resource should be available from this time
availability_end_date string 11:00

Resource should be available till this time
additional_restypes_reqd boolean false

Additional Resource types would be shared for this category if this parameter is true
res_sch_category string DOC

Resources must belong to this category
